OpenGL Playground

      This is not really a project, but more of a place where I can learn OpenGL and grow.

      The thumbnail above is the current state of the project

      As you can see, so far I have

      • Rendered a cube using OpenGL
      • Created a Graphical User Interface to easily change settings without compiling the code
      • Added lighting

      All of it is in C++!

      While this may seem trivial at first, it isn't as just the initialization file is already 400 lines of code! 400 lines of code just to boot a simple window game.

      This is beacause I am coding most of this from scratch, including the lighting algorithm.

      Libraries/APIs used

      • Win32 API
      • OpenGL
      • Dear Imgui
