C / C++

      C

      C is an old programming language, but still really powerful.

      As a compiled language, it can:

      • give the programmer low level access to computer resources
      • make really fast apps
      • Allow for development on many different platforms/architectures

      C++

      C++ is based on C and adds additional features to C. It supports C operations but adds features like:

      • object oriented programming
      • templates
      • the Standard Template Library(STL) for common data structures
      • function overloading
      • operator overloading
      • and many more small features
