      How it is organized

      As seen on the Github, each page gets its own folder under a very similar file hierarchy found in the navigation bar. Each page has its own folder to hold the images, icons, and other files associated with that page.

      However, all pages have some similarities. For example the purple navigation bar.

      I could just copy and paste the html for the navigation bar into all files, but this has a major problem.

      If I ever want to add a project and/or update the navigation bar, I would have to go in and manually change the code for the navigation bar for EVERY SINGLE FILE, not very time effective.

      To solve this, I create one html file with all the HTML for the navigation bar and then use Javascript to inject that html into the page. 

      
          function navBarMQ() {
            if (window.innerWidth <= maxWidth) {
              navBar.innerHTML = '';
              navBar.style.margin = '0';
              main.style.width = 'calc(100% - 30px)';
              main.style.marginLeft = '15px';
            } else {
              navBar.innerHTML = navBarHTML;
              navMenu.innerHTML = ''
              navBar.style.margin = null;
              main.style.width = null;
              main.style.marginLeft = null;
            }
            if (window.innerWidth <= maxWidth && !topBarMenuState) {
              topBar.appendChild(menuIcon)
              topBarMenuState = true;
            } else if (window.innerWidth > maxWidth && topBarMenuState){
              topBar.removeChild(menuIcon);
              topBarMenuState = false;
            }
          }

      navBarMQ() is a funcion that is called on page load, and whenever the viewport is resized
