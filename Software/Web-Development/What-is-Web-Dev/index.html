      How are websites developed?

      All websites have a frontend, what the user sees, and a backend, which provides the user with the content (host).

      The frontend is often designed using HTML, CSS, and Javascript.

      A good way to think about what each one does:

      "HTML is the noun, CSS is the adjective, and Javascript is the verb." - From a comment on Youtube
