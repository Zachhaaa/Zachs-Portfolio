Git / Github

      What is Git?

      When programming, we need a way to save different versions of our repository to keep track of files and undo changes if necessary.

      Software that does this is called version control software, and Git is the by far the most popular one.

      What is Github?

      Using git we can publish projects to github. This is where organizations publish their open source projects.

      For example, you can find the source code for the Linux kernel on this site. People can clone the repository and make changes to the code.
