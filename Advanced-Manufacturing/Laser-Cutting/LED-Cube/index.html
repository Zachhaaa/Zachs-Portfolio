LED Cube

      Process

      1. Got the outline of the box from Makercase
      2. Added the honeycomb pattern using inkscape
      3. Glued paper to the insides so the light from the LEDs diffuses
      4. Soldered the electronics together which included the following components
        • LEDs
        • Resistors
        • Switch
        • Power jack
      5. Hot glued the box together
