Titans of CNC 4m

      What is it?

      This was my first CAM and CNC machining project. I followed a tutorial using this website, and I machined it using the HAAS CNC machine at my school robotics lab.

      CAD

      The part has no practical purpose. However, it is a great part to learn CAM due to features that require different tools and machining strategies.

      CAM

      Is composed of two setups and uses the following tools

      • Face mill
      • 3/8" End mill
      • 3/4" Key cutter
      • 3/8" Chamfer mill
      • Drill
      • 10-32 Tap
